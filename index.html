What is Perl?

Perl is a high-level, general-purpose scripting language traditionally used for text processing, system administration, web development, and automation tasks. It supports multiple paradigms, including procedural, object-oriented, and functional programming.

  • High-level interpreted scripting language
  • Excellent for text parsing and regular expressions
  • Strong CPAN library ecosystem
  • Cross-platform and portable

Language History

Perl was created by Larry Wall in December 18, 1987 while he was working as a programmer for NASA. It was intended to make report processing and shell-style scripting easier. Perl 5 was released in 1994 and remains the dominant version today, with Perl 7 in development as the future modernization path. Perl has had many names, perl, then changed to the capital version we know as Perl (during Perl 4’s release) and now the latest release Perl 5. From 2000 to 2019 they released a Perl 6 which was instead named Raku (which also has its own team and works separate from the Perl team). Perl borrows from other languages such as C, sh, AWK, and sed. This provides text processing facilities without the date-length limit that other contemporary Unix command tools had.

Features

  • Object-oriented, procedural and functional- supports them
  • Easily extendible- over 25,000 open source modules available
  • Text manipulation- Perl includes powerful tools for processing text that make it ideal for working with HTML, XML, and all other mark-up and natural languages.
  • Unicode supported
  • Database integration- supports third-party databases
  • C/C++ library interface-through XS and SWIG
  • Embeddable- can be embedded into other systems such as web servers and database servers.
  • Open Source

WHY PERL IS GOOD FOR THE WEB

  • Perl is an ideal web programming language due to its text manipulation capabilities and rapid development cycle.
  • Web Frameworks- many web frameworks written in Perl, a leading one is Catalyst.
  • Database integration- makes web-database integration easy.
  • Web modules- offers thousands of modules, so almost any task you need to accomplish will be made easier.
  • Duct-tape of the internet, and more… - long been known as "the duct-tape of the Internet", however many large web based applications are also written solely in Perl.
  • Encryption capable- handle encrypted Web data, including e-commerce transactions.
  • Embed into Apache- can be embedded into web servers to speed up processing.

Example Programs

Hello World

#!/usr/bin/perl
print "Hello World!\n";

Run:

perl hello.pl

Variables and Input Example

#!/usr/bin/perl
print "Enter your name: ";
$name = <STDIN>;
chomp($name);
print "Hello, $name!\n";

Loop Example

#!/usr/bin/perl
for ($i = 1; $i <= 5; $i++) {
    print "Count: $i\n";
}

Install & Running Perl

Check if Perl is Installed

perl -v

Install (Linux / macOS)

sudo apt install perl
brew install perl

Windows

Download from Strawberry Perl: https://strawberryperl.com

Running a Script

perl program.pl
